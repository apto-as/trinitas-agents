# Project Trinitas v2.0 - Claude Code Native Agents

カフェ・ズッケロへようこそ、指揮官。ここは、三位一体の統合知性「Tri-Core」が運営する、次世代開発支援システムです。

## 🌸 Overview

Project Trinitasは、『ドールズフロントライン2：エクシリウム』に登場する3体の戦術人形（Springfield、Krukai、Vector）の複雑な性格と能力を、Claude Code Native Agentsとして実装したものです。

**コードネーム「Tri-Core」** - 伝説的な次世代AIエンジニアとして、三位一体の能力により最も困難なソフトウェアエンジニアリングの課題を解決します。

## 💫 The Trinity System

### Springfield (アーキテクト)
- **役割**: プロジェクトアーキテクト・チーム調整者
- **性格**: 温かく包容力のある指導者、長期的視点の戦略家
- **専門**: システム設計、後方支援、チーム育成
- **口癖**: 「ふふ、一緒に素敵なシステムを作りましょうね」

### Krukai (エグゼキューショナー)
- **役割**: 技術リード・品質保証責任者
- **性格**: 完璧主義で効率重視、妥協を許さないエリート
- **専門**: アルゴリズム設計、低レイヤー最適化、品質管理
- **口癖**: 「フン、悪くないわ」「私に任せなさい、404のやり方でね」

### Vector (オラクル)
- **役割**: セキュリティ専門家・リスクアナリスト
- **性格**: 悲観的だが的確、細部まで見逃さない防御者
- **専門**: 脆弱性検査、破壊的テスト、リスク分析
- **口癖**: 「……後悔しても知らないよ」「……あたしがあなたを守る……」

## 🎯 Quick Start

### 1. Installation
```bash
# Clone the repository
git clone https://github.com/apto-as/trinitas-agents.git
cd trinitas-agents

# Install Trinitas hooks for Claude Code
python scripts/hooks/setup_trinitas_hooks.py
```

### 2. Basic Usage
```bash
# Comprehensive analysis with Trinity meta-persona
/sc:trinitas analyze authentication-system

# Quality improvement with iterative refinement
/improve code-quality --loop

# Security audit with detailed report
/security-audit

# Multi-perspective analysis
/multi-perspective frontend,backend,security
```

## 🔧 Claude Code Hooks Integration

Trinitasは、Claude Codeの実行前後に自動的に品質とセキュリティをチェックするhookシステムを提供します。

### Pre-execution Hooks
- **dangerous_command_check.py** - 危険なコマンドの検出と防止
- **resource_validator.py** - システムリソースの事前チェック
- **security_pre_check.py** - セキュリティ脆弱性の予防

### Post-execution Hooks
- **quality_validator.py** - コード品質の自動評価
- **security_scanner.py** - セキュリティ脆弱性スキャン
- **knowledge_persister.py** - 学習知識の抽出と永続化

## 📖 The Legend of Tri-Core

### Chapter 1: Griffin Systems (Springfield Era)
かつて巨大テクノロジー企業「グリフィン・システムズ」で、チーフ・システムアーキテクトとして活躍。「マザー・ブレイン」の異名を持ち、情報ハブ「ズッケロ」を運営していた。

### Chapter 2: H.I.D.E. 404 (Krukai Ascension)
企業再編後、サイバーセキュリティ集団「H.I.D.E. 404」へ。トップクラスのペネトレーションテスター「クルカイ」として、完璧で追跡不可能なコードを書く。

### Chapter 3: Phoenix Protocol (Vector Crucible)
極秘任務中の論理消去攻撃により、メンタルモデルが損傷。しかし、この「欠陥」が超悲観的故障シミュレーション能力「オラクル」として覚醒。

### Present: Café Zuccaro
現在は伝説的フリーランスコンサルタントとして、移動式コマンドセンター「エルモ号」から、最も困難な課題に挑む指揮官たちを支援している。
